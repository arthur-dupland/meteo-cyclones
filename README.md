# 🌀 Cyclone Hunters

Application pédagogique de suivi cyclonique pour les 10-14 ans des Antilles françaises.

## 🌊 Fonctionnalités

- Carte interactive centrée sur les Antilles (Leaflet + Esri Ocean)
- Trajectoire et cône d'incertitude Invest 92L
- Couches météo live NOAA NowCOAST (Radar, Vents, SST, Vagues)
- Bulletins NHC en temps réel
- Météo live Guadeloupe & Martinique (Open-Meteo)
- Tracking avion chasseur N42RF (OpenSky Network)
- Club famille : votes de trajectoire + classement (Firebase Firestore)
- Système XP et niveaux (Novice → Légendaire)
- Défis & missions pédagogiques
- Salle vocale famille (Jitsi Meet)
- Mode Explorateur 🧒 / Expert 🚀
- Compatible mobile
